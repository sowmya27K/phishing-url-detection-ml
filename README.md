# Phishing URL Detection using Machine Learning

## Project Overview
This project is part of my MCA curriculum. It focuses on detecting phishing websites
using machine learning techniques by analyzing URL-based features.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Machine Learning
- Jupyter Notebook

## Project Workflow
1. Data collection
2. URL feature extraction
3. Data preprocessing
4. Model training
5. Phishing URL prediction

## Objective
To help identify malicious URLs and reduce cyber security threats.
